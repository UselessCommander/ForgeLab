# Sådan kører du projektet lokalt

## Hurtig Start

### Metode 1: Brug start script (Windows)
Dobbeltklik på `start-server.bat` filen

### Metode 2: Brug npm
```bash
npm start
```

### Metode 3: Brug PowerShell script
```powershell
.\start-server.ps1
```

## Efter serveren starter

1. **Åbn browseren** og gå til:
   ```
   http://localhost:3000
   ```

2. **For QR-koder fra telefon:**
   - Brug IP-adressen der vises i konsollen (f.eks. `http://192.168.50.137:3000`)
   - Telefon og computer skal være på samme WiFi

## Sider

- **Hoved side:** http://localhost:3000/
- **Dashboard:** http://localhost:3000/admin.html

## Stop serveren

Tryk `Ctrl + C` i terminalen hvor serveren kører.
